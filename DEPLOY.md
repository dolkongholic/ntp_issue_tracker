# 배포 (포트포워딩으로 외부 공개)

## 실행 방법

```bash
npm run build   # next build (프로덕션 빌드)
npm run start   # next start -p 5001 (프로덕션 서버, 5001 포트)
```

`npm run dev`는 nodemon + hot reload가 붙은 **개발용**입니다. 외부에 열 때는 사용하지 마세요.
빌드 없이 dev 서버로 포트포워딩하면 코드 변경마다 재시작되고 최적화되지 않은 빌드가 노출됩니다.

## 포트

내부 서비스 포트: **5001** (`dev`, `dev:next`, `start` 모두 통일)

공유기/서버 포트포워딩 설정 시 내부(로컬) 포트를 `5001`로, 외부 포트는 원하는 번호로 지정하면 됩니다.

## 체크리스트

- [ ] `npm run build` 성공 확인
- [ ] `npm run start`로 로컬에서 `http://localhost:5001` 정상 동작 확인
- [ ] 공유기 포트포워딩: 외부 포트 → 내부 IP:5001
- [ ] 방화벽에서 5001 포트 인바운드 허용
